Building a NAS & Linux Server from Old PC Parts

Why Build a NAS?

A Network Attached Storage (NAS) server lets you share files, run backups, and host media from anywhere on your home network — all using free open-source tools.

What I Used

  • Old desktop PC (quad-core, 4GB RAM, 2x 1TB HDD)
  • Ubuntu Server / Linux Mint
  • Samba (file sharing)
  • OpenMediaVault (optional web-based interface)
  • FreeNAS / TrueNAS (for ZFS storage, snapshots, and enterprise-grade features)

Steps

  1. Installed Ubuntu Server, Linux Mint, or FreeNAS via USB
  2. Formatted and mounted HDDs using built-in partitioning tools
  3. Installed Samba: sudo apt install samba or configured CIFS shares in FreeNAS
  4. Configured /etc/samba/smb.conf or FreeNAS GUI with shared folders
  5. Set permissions and enabled remote access
  6. Optional: Installed OpenMediaVault for web UI management (Linux only)

Optional Features

  • Used rsync for backups
  • Ran a Minecraft server in background
  • Used it as a torrent box with Transmission
  • Mounted shares on Windows and Android
Disclaimer: Always secure your NAS with user authentication and firewalls. Local network use recommended.
