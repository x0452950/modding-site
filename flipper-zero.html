Flipper Zero Projects & Tools

What is Flipper Zero?

Flipper Zero is a portable multi-tool for pentesters, hackers, and hardware tinkerers. It supports RF, IR, RFID/NFC, GPIO, and more — all from a tiny Tamagotchi-style device.

Tools I Use

  • Official Flipper Firmware + QFlipper
  • Unleashed/Custom Firmware for advanced features
  • Flipper Apps + Plugins (badusb, signal replay)
  • SubGHz signal capture
  • U2F dongle for 2FA testing

Project Ideas

  • Replay garage remotes with SubGHz
  • Test RFID badge access
  • Use GPIO to trigger relays
  • Brute force PINs (for fun, not crime!)
  • IR blast to control TVs in public places
