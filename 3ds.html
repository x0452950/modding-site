3DS Modding Guide (2025 Update)

Disclaimer: This guide is for educational purposes only. Do not use custom firmware (CFW) to run pirated content. Modding your 3DS may void your warranty.

What Is 3DS Modding?

Modding a Nintendo 3DS allows you to unlock new features like homebrew apps, emulators, custom themes, save file backups, and more. You’ll install custom firmware (CFW), typically using a free tool called Boot9Strap or Luma3DS.

Is Your 3DS Moddable?

Model Supported Firmware Modding Method
Old/New 3DS/2DS 11.17 or below SafeB9SInstaller + Luma3DS
2DS XL 11.17 or below Same as above

Check your system version: Go to System Settings → it's shown on the bottom-right corner of the top screen.

What You’ll Need

  • A 3DS/2DS with firmware 11.17 or lower
  • A microSD card (at least 4GB)
  • A computer (Windows/macOS/Linux)
  • Internet access on your 3DS
  • Files from: 3ds.hacks.guide

Step-by-Step Installation

  1. Prep Your SD Card: Format to FAT32. Download the latest versions of Luma3DS, SafeB9SInstaller, and boot9strap. Place the files on the SD card root.
  2. Install PicHaxx: Download the save file. Open the Nintendo 3DS Camera and scan the PicHaxx QR code. Follow prompts to open the Homebrew Launcher.
  3. Run SafeB9SInstaller: Launch from Homebrew. This installs Boot9Strap and Luma3DS CFW. You’ll see the Luma3DS config screen after reboot.
  4. Finalize Setup: Enable options like "Show NAND" and "Enable game patching". Press Start to reboot.

What Now?

  • Launch Homebrew Launcher from the Home Menu
  • Install FBI for homebrew apps
  • Use Checkpoint for save backups
  • Install custom themes and emulators

Common Issues

  • Black screen after modding: Hold Start + Select + Power to re-enter Luma config
  • Homebrew Launcher won’t boot: Recheck your SD card files and formatting
