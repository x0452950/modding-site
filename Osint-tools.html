OSINT Tools Directory

1. General Search & Meta-Search Engines

  • Google Advanced Search: Filter results by date, file type, or site-specific searches.
  • DuckDuckGo: Privacy-focused search engine.
  • Startpage: Delivers Google results without tracking.
  • Yandex: Russian search engine with strong image/text search capabilities.

2. Specialized Search Tools

  • Shodan: Search internet-connected devices (IoT, servers, cameras).
  • Censys: Similar to Shodan, for network device discovery.
  • Wayback Machine (Archive.org): View archived versions of websites.

3. Social Media & Username Tools

  • Social Searcher: Cross-platform social media keyword/search monitoring.
  • Osint.link: Aggregator of social media search tools.
  • Sherlock (GitHub): Check username availability across platforms.
  • TinEye / Yandex Images: Reverse image search to find image sources.

4. Domain & IP Investigation

  • WHOIS Lookup (ICANN): Check domain registration details.
  • DomainTools: Historical DNS records, WHOIS, and threat intelligence.
  • MXToolbox: Diagnose email/DNS issues and blacklist status.
  • AbuseIPDB: Report and check IP addresses for malicious activity.

5. Email & Phone Number Tools

  • Hunter.io: Find email addresses associated with a domain.
  • Phonebook.cz: Discover phone numbers linked to a domain (by Hunter.io).
  • Truecaller: Identify unknown phone numbers (global database).

6. Public Records & Government Data

  • EDGAR (SEC): U.S. Securities and Exchange Commission filings.
  • PACER: U.S. federal court records (requires account).
  • OpenCorporates: Global database of company registrations.
  • World Bank Open Data: Global economic/social development datasets.

7. Geospatial & Mapping Tools

  • Google Earth Pro: High-resolution satellite imagery.
  • Sentinel Hub: Access satellite imagery (e.g., ESA Sentinel missions).
  • Flightradar24: Real-time flight tracking.

8. People & Identity Verification

  • Pipl: Search for people using email, phone, or name.
  • IntelTechniques Search Tools: Aggregated OSINT tools by Michael Bazzell.

9. Threat Intelligence & Cybersecurity

  • VirusTotal: Analyze suspicious files/URLs for malware.
  • Hybrid Analysis: Sandboxed malware analysis.
  • DeHashed: Search breached credentials and leaked databases.

10. News & Media Monitoring

  • Google News Archive: Historical news articles.
  • MuckRock: Request and analyze public records.
