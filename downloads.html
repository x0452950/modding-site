Downloads & Tools I Use

Modding Tools

  • Odin – Samsung flashing tool (Windows)
  • ADB & Fastboot – Android debug tools
  • TWRP Recovery – Custom recovery for Android
  • SuperSU / Magisk – Android root access tools
  • FBI (3DS) – Install .CIA files
  • Luma3DS – Custom firmware for Nintendo 3DS
  • PRO-CFW (PSP) – Custom firmware for PSP
  • USB Loader GX (Wii) – Load games from USB
  • OpenMediaVault – NAS software

Useful Linux Tools

  • Etcher – USB image flashing
  • GParted – Partition manager
  • Rsync – File sync for backups
  • Samba – Network file sharing

Bootable USB & Disk Tools

  • Rufus – Create bootable USB drives from ISO files
  • Ventoy – Boot multiple ISOs from a single USB
  • Etcher – Flash OS images to USB/SD cards
  • Universal USB Installer – Create bootable USBs for Linux
  • Clonezilla – Disk cloning and backup

System & Package Management

  • Ninite – Auto-installer for Windows apps
  • Chocolatey – Command-line package manager for Windows

Encryption & Security Tools

  • Kleopatra (Gpg4win) – Encrypt and sign files and emails
  • OpenSSL – Toolkit for SSL/TLS and cryptography

Reverse Engineering & Debugging

  • IDA Free – Disassembler for reverse engineering
  • Ghidra – Reverse engineering suite by NSA
  • x64dbg – Windows debugger

Networking Tools

  • Wireshark – Packet capture and analyzer
  • Nmap – Network scanning and security auditing

Utilities & Misc

  • 7-Zip – Archiving tool supporting many formats
  • Notepad++ – Lightweight code/text editor
  • Visual Studio Code – Advanced code editor with extensions
  • Revo Uninstaller – Deep-clean app removal
  • Everything – Lightning-fast file search
  • CPU-Z – System info utility
  • HWMonitor – System temperature and voltage monitor
  • VirtualBox – Run other OSes in a virtual machine
  • RetroArch – Multi-platform emulator
  • Frida – Dynamic instrumentation toolkit for apps
