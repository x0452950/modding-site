PS Vita Custom Firmware Guide

What I Did

I softmodded my PS Vita to run backups, emulators, and homebrew. I used a combination of H-encore, Enso, and SD2Vita to fully unlock the system. This guide is a mix of my experience and proven methods used by the community.

Tools I Used

  • H-encore – Used to exploit firmware 3.60–3.68 and enable homebrew access.
  • Enso – Made the CFW permanent (auto-runs on boot) on supported firmwares.
  • Adrenaline – Enabled a full PSP emulator on Vita for running PSP games and plugins.
  • SD2Vita – Let me use a microSD card as internal storage via the game card slot.

Custom Firmware Installation Techniques

  • Firmware 3.60: Best firmware. Use HENkaku with henkaku.xyz directly in browser. Then install Enso for permanent CFW.
  • Firmware 3.65: Update from 3.60 and install Enso-compatible H-encore. Requires some manual file prep.
  • Firmware 3.67 / 3.68: Use H-encore to gain temporary CFW each reboot. Install VitaShell, run homebrew.
  • Firmware 3.69+: Downgrade using modoru to 3.60/3.65 to install Enso. This was the route I had to take.

Homebrew I Installed

  • VitaShell – File manager and homebrew installer
  • Adrenaline – Full PSP firmware layer inside Vita
  • PKGj – FreeShop-style app for downloading games (homebrew-only use!)
  • Autoplugin – Quick install of essential plugins like overclocking, SD2Vita, battery %, etc.

Notes from My Setup

  • I had to downgrade from 3.73 to 3.65 using Modoru — risky but worth it to get Enso.
  • SD2Vita required setting up storage_config.txt correctly or it wouldn’t boot.
  • Enso made my setup truly bootable without rerunning H-encore every time.
