Voice AI Assistant

Create a private and responsive voice assistant on Android using ChatGPT API, ElevenLabs for voice, and Tasker for automation.

Features

  • Voice input captured on Android
  • Responses generated by ChatGPT API
  • Optional ElevenLabs voice output
  • Real-time, offline-triggered personal assistant

Step-by-Step Setup

  1. Install Required Apps:
    • Tasker (Play Store)
    • AutoTools (for JSON parsing)
    • AutoNotification / AutoVoice (optional for triggering)
  2. Generate Your API Key:
    Visit OpenAI, log in, and create a secret API key.
  3. Create a Task in Tasker:
    Name it Ask ChatGPT. Add the following actions:
    • 1. Get Voice Input: Use “Get Voice” action → Save to %Voice
    • 2. HTTP POST to ChatGPT:
      • URL: https://api.openai.com/v1/chat/completions
Headers:
  Authorization: Bearer YOUR_API_KEY
  Content-Type: application/json
Body:
{
  "model": "gpt-3.5-turbo",
  "messages": [
    {"role": "user", "content": "%Voice"}
  ]
}
    • 3. Extract ChatGPT Response: Use AutoTools JSON Read → path: choices[0].message.content
  4. Speak the Reply:
    Use Tasker's built-in "Say" or AutoVoice. For natural voice:
    • Send text to ElevenLabs API
    • Download audio and play via Tasker's "Play Music" or "AutoTools Media"
  5. Optional Triggers:
    Launch via:
    • Home screen shortcut
    • Notification button (AutoNotification)
    • Hotword using AutoVoice

Tips & Notes

  • Test your ChatGPT body using Postman first if needed
  • Use %Voice variable dynamically to improve flow
  • Cache responses locally for faster repeats
  • Throttle or filter requests to avoid spam calls to APIs
