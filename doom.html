Install Rockbox on Your Sansa MP3

This guide shows how to install Rockbox firmware and run DOOM on a Sansa MP3 player.

How to Install Rockbox

  1. Go to rockbox.org/download.
  2. Choose your Sansa model (e.g. e200, Clip+, etc.).
  3. Download the Rockbox Utility for your OS.
  4. Connect your player via USB and run the utility.
  5. Select your device and install the complete package.
  6. Safely eject and reboot your player â€” you should see the Rockbox boot screen.

Run DOOM on Rockbox

  1. Download the shareware version of DOOM1.WAD.
  2. Create a folder on your Sansa at /.rockbox/doom/.
  3. Place DOOM1.WAD inside the doom folder.
  4. Disconnect your player and reboot into Rockbox.
  5. From the main menu, go to Games > Doom.
  6. Select DOOM1.WAD and enjoy!
