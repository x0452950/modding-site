Raspberry Pi Setup & Projects

Why I Use Raspberry Pi

The Raspberry Pi is perfect for DIY tech experiments — small, affordable, and surprisingly powerful. I've used it as a server, emulator box, and more.

Tools & Accessories

  • Raspberry Pi 3/4
  • microSD card (16GB+)
  • Power supply & case
  • HDMI cable, monitor, keyboard (for first setup)
  • Image writer like Etcher

Projects I've Built

  • RetroPie – Classic game emulator box for SNES, GBA, PS1, etc.
  • Pi-hole – Network-wide ad blocker
  • Home NAS – Basic file server with Samba
  • VPN Server – Private Pi VPN using OpenVPN
  • Media Center – Kodi-based local video and streaming hub

Setup Steps (Example: RetroPie)

  1. Flashed RetroPie image to SD card using Etcher
  2. Booted Pi, configured controller
  3. Copied ROMs via USB or network
  4. Connected to Wi-Fi and updated packages
Disclaimer: Emulators are legal, but only use ROMs you legally own.
