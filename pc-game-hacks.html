PC Game Hacks & Save Editors

Overview

This page lists my favorite tools for modding single-player PC games, including trainers, cheat engines, and save editors. Use responsibly and offline.

Cheat Engine

Cheat Engine lets you scan and modify memory while the game is running. You can freeze values, increase currency, or enable god mode.

  • Website: cheatengine.org
  • Use with tables for specific games (e.g., fearlessrevolution.com)
  • Always run as Administrator for best results

Universal Steam Game Mods & Hacks

  • Skyrim: Console commands like tgm, player.additem, mods via Nexus
  • Fallout: Similar to Skyrim, console plus Nexus mods
  • Cyberpunk 2077: Use Cyber Engine Tweaks for scripting and item editing
  • The Witcher 3: Debug console enabled via mod, commands like additem and god
  • GTA V: Use Menyoo or other SP trainers (not online!)
  • Stardew Valley: Save editors and name exploits

Gibbed's Borderlands 2 Save Editor

Customize your Borderlands 2 character, inventory, mission status, and more with this save editor.

  • Gibbed Save Editor on GitHub
  • Supports PC, Xbox, and PS3 save conversion
  • Backup your save files before use

Trainers

Trainers are standalone programs that enable cheats like unlimited ammo, health, or money with hotkeys. Great for quick fun in single-player games.

  • WeMod: Easy trainer platform with support for hundreds of games
  • MrAntiFun: Independent trainer maker with tons of supported games
  • FLiNG Trainers: Known for high-quality, regularly updated trainers
