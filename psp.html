Modding the PSP with Custom Firmware (CFW)

Why Mod the PSP?

Modding the PSP opens it up to homebrew games, emulators, custom themes, and playing ISO/CSO backups.

Tools Used

  • PSP 1000/2000/3000 or PSP Go
  • Memory Stick Duo or internal storage (PSP Go)
  • Firmware 6.60 or 6.61
  • Pro CFW (e.g. PRO-C2)

Steps

  1. Updated PSP to 6.60 or 6.61 official firmware
  2. Copied PRO CFW files to PSP `/PSP/GAME/` folder
  3. Ran PRO Update from the PSP Game menu
  4. Installed CFW and optionally made it permanent with CIPL Flasher (if supported)

What You Can Do Now

  • Play ISO/CSO backups from memory stick
  • Install homebrew games and apps
  • Use emulators for NES, SNES, GBA, etc.
  • Apply custom themes and plugins
